## Deskripsi
Automation testing untuk fitur login pada OrangeHRM menggunakan Cypress.

Website:
https://opensource-demo.orangehrmlive.com/web/index.php/auth/login

## Test Case
1. Logo tampil
2. Input username
3. Input password
4. Login berhasil
5. Forgot password
6. Validasi field kosong
7. Password hidden
8. Form login tampil
9. Layout login tampil
10. Login dengan tombol enter

Semua test case berhasil PASS.
